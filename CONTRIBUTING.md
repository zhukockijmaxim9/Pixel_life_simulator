# Contributing

Спасибо, что хотите помочь проекту Pixel Life Simulator.

## Быстрый старт для разработки

### Требования

- Flutter SDK установлен и доступен в PATH

### Установка зависимостей

```bash
flutter pub get
```

### Запуск

```bash
flutter run
```

### Анализ и тесты

```bash
flutter analyze
flutter test
```

## Как предлагать изменения

1. Создайте ветку от `main`.
2. Делайте небольшие, логически цельные коммиты.
3. Убедитесь, что проект собирается и проходит `flutter analyze`.
4. Откройте Pull Request с описанием:
   - что изменилось и зачем,
   - как это проверить вручную.

## Кодстайл

- Следуем линтам из `analysis_options.yaml`.
- Публичные сущности документируем через DartDoc (`///`) только там, где это добавляет смысл.
- Избегаем “комментариев-очевидностей”.

## Контент и баланс

Данные игры лежат в `lib/data/`. Если добавляете новые события/профессии/цели:

- сохраняйте уникальность `id`,
- тестируйте сценарии “денег не хватило” и “победа/поражение”,
- при необходимости обновляйте `docs/gameplay.md` и `docs/assets.md`.

## Сообщения об ошибках

В issue/PR полезно приложить:

- шаги воспроизведения,
- фактический/ожидаемый результат,
- платформу (Android/iOS/Web/Windows),
- лог из консоли (если есть).
