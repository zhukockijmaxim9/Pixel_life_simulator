# Pixel Life Simulator (Pixel Purse)

Небольшой симулятор финансовой жизни в пиксельной стилистике: распределяйте зарплату по категориям, проживайте события месяца, повышайте доход через карьеру и курсы, копите на цель и тратьте баллы в магазине мерча.

## Возможности

- **Бюджетирование**: распределение денег между кошельком, “отложенным”, обязательными расходами и целью.
- **Цели**: накопление на выбранную цель и получение игровых баллов за успешный месяц.
- **События**: случайные/добровольные события, платежи, образовательные квизы.
- **Карьера**: выбор работы, повышение уровня, открытие новых вакансий через опыт/курсы.
- **Магазин**: покупка мерча за игровые баллы.
- **Сохранение прогресса**: локально на устройстве через `shared_preferences`.

## Технологии

- **Flutter / Dart**
- **State management**: Provider (`ChangeNotifier`)
- **Локальное хранилище**: `shared_preferences`
- **UI**: Google Fonts (Press Start 2P), тёмная тема

## Быстрый старт

### Требования

- **Flutter SDK**: установлен и доступен в PATH
- **Dart SDK**: ставится вместе с Flutter

### Установка

```bash
flutter pub get
```

### Запуск

```bash
flutter run
```

Пример для веба:

```bash
flutter run -d chrome
```

### Проверки и тесты

```bash
flutter analyze
flutter test
```

## Структура проекта

- `lib/`
  - `main.dart` — точка входа, роутинг, тема
  - `app_state.dart` — `GameState`: основная игровая логика, расчёты, сохранение/загрузка
  - `data/` — статические данные (профессии, цели, курсы, события, мерч)
  - `domain/managers/` — менеджеры доменной логики (`FinanceManager`, `CareerManager`)
  - `models/` — модели данных (Job/Goal/Event/Merch/Course и т.д.)
  - `screens/` — экраны приложения
  - `widgets/` — переиспользуемые UI-компоненты
- `assets/` — шрифты/иконки/медиа
