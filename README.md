# Pixel Life Simulator 🎮💰

## ✨ Основные возможности

- 📈 **Управление бюджетом**: Распределяйте зарплату между обязательными и переменными расходами.
- 🎯 **Финансовые цели**: Копите на отпуск, новый гаджет или формируйте «подушку безопасности».
- 🛍️ **Внутриигровой магазин**: Обменивайте накопленные баллы на мерч.
- 🧠 **Образовательный блок**: Изучайте финансовые термины и принципы прямо в процессе игры.
- ⚡ **Динамические события**: Сталкивайтесь с непредвиденными ситуациями, требующими быстрых финансовых решений.

---

## 🛠 Технологический стек

- **Язык**: Dart
- **Фреймворк**: [Flutter](https://flutter.dev/) (3.10.0+)
- **Управление состоянием**: [Provider](https://pub.dev/packages/provider)
- **Шрифты**: [Google Fonts](https://pub.dev/packages/google_fonts)

---

## 📂 Структура проекта

- `lib/` — Основной исходный код
  - `screens/` — Экраны приложения (Главное меню, Планирование, Магазин и др.)
  - `widgets/` — Переиспользуемые компоненты интерфейса
  - `app_state.dart` — Логика управления состоянием игры
- `assets/` — Изображения, шрифты и другие ресурсы (при наличии)

## 🚀 Инструкция по запуску

Для запуска проекта убедитесь, что у вас установлен [Flutter SDK](https://docs.flutter.dev/get-started/install).

1. **Клонируйте репозиторий**:
   ```bash
   git clone https://github.com/zhukockijmaxim9/Pixel_life_simulator.git
   cd Pixel_life_simulator
   ```

2. **Установите зависимости**:
   ```bash
   flutter pub get
   ```

3. **Запустите приложение**:
   
   Вы можете запустить проект на подключенном устройстве, эмуляторе или в браузере:
   ```bash
   flutter run
   ```

4. **Запуск в браузере (Chrome)**:
   ```bash
   flutter run -d chrome
   ```

5. **Запуск тестов**:
   ```bash
   flutter test
   ```
